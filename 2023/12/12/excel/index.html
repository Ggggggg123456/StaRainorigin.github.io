Excel函数使用

2023/12/12 Excel

函数

函数的形式：=函数名(参数们)

函数存在的本意其实是为了更方便的处理数据，
如下面的场景下，第一行用函数求了各项目的总和，如果不用函数，下面的每个数据经过修改，这个总和我们都要重新算一遍，如果用函数，这个总和就不用动了，因为下面的值改变过后，总和会自己重新求一边。

理解

我们学的编程语言里面也有函数，差不多的东西，我就简单说一下我自己的理解吧。
有些概念是我自己编的，目的是方便理解

关于函数

跟数学上的函数不同，函数像是一个进行加工的工厂，你给它原材料，它给你成品。然后工厂有不同的类型，有编织衣服的有做不锈钢盆的……
类比函数的话，参数就是原材料，函数名就是它是一个加工什么的工厂，函数生成的结果就是成品。
这么看=编织(毛线, 针, ...)，就是 把毛线，针给给编织工厂，工厂最终会给你做出衣服来。

关于参数

参数的==数量==可以不止一个，比如 编织 函数，就需要 毛球，针，...
参数的==数值==可以不一样，如 毛线， 如果给它黄色毛线 ，就会织出黄色衣服，给蓝色毛线，就会织出蓝色衣服。

excel中，一般参数都是写某个单元格，然后传入的是单元格里面的内容
你们书上写的函数中的参数，一个参数对应的值可能是一个值，也可能是多个值，但是都算作一个参数，参数之间用逗号 ‘,’ 分隔开
你输入单元格，比如: “A1、E4、B5”, 这种就是把==一个==单元格里面的值传入，

如果是这种形式 “A1: A12 、A3: G3”, 这种就是传入==多个==单元格里的值，如”A1: A12”，就是指的竖着一列中的几个值，”A3: G3”就是横着一行中的几个值。它们虽然是多个数，但是都算作==一个参数==

关于函数结果

=函数名(参数们)
你在一个excel单元格里输入一个函数，不管你用的什么函数，传入的什么参数，最后都会生成一个或一些值，这就是函数产生的结果。
函数产生的结果也可以当作参数传入其他函数，初学者也许不太好懂？下面有解释

函数嵌套

不要被名字唬到了
假如你在一个单元格中输入=编织(毛线, 针, ...)，那么单元格中最终显示的就是一件衣服。你把这个单元格直接当成一件衣服就好了，
假如说再有个函数叫 =销售(衣服), 此时这个单元格的值 衣服 虽然是通过 编织 函数得出来的，但是它也是一个值，跟你直接在单元格中输入这件 衣服 是一样的，所以它也可以作为 =销售(衣服) 的参数传入到函数中去。

插入函数

其实就是在一个单元格中输入 =函数名(参数们) ，假如你想在一个单元格里面求和，就在这个单元格中输入 =SUM(求和范围) 。你们书上写的Shift+F3，或者是编辑栏上的点图标的方式，本质上也会生成这么个东西，只是都变成用鼠标点击操控，方便一些没学过计算机编程语言的人理解这些东西，你用你喜欢的方式就好。

就是做一些数学计算的时候用的，取整、四舍五入、求和、加减乘除，只不过加减乘除可以直接用 + - * / 符号完成，这里除法也给了个函数，我没深入研究跟直接用 / 符号有什么区别，我用着都一样

统计函数

就是做一些关于统计方面的事情用的，比如 计数，平均值，最大最小值等，你们如果学统计学就会发现统计学里面喜欢研究的就是这些东西，还有方差之类的。

COUNTIF 后面再说

日期函数

就是一些处理日期相关内容的函数

其中 DATE 函数就是你输入年月日的数值，它给你生成一种 规范了书写格式 的日期，这么做的原因是
没有人工智能的情况下，计算机对文字的处理比较笨，它只能处理特定格式的数据，这个 DATE 函数其实就是为了把日期转换成计算机能处理的格式，方便后面能把这个数据放在其他函数(比如MONTH) 里面使用

逻辑函数

一些逻辑学相关的函数，真假、IF、与或非
然后我觉得主要是还是 IF，你可以以 IF 为”骨架”来理解这部分，其他的”真假、与或非”，都是为这个 IF 服务的

真与假

真假就跟真命题假命题一样

真：如 2>1, 4<5, 5=5，这些条件是满足的，就是真的，我们叫TRUE
假：如 2<1, 4>5, 5≠5，这些条件都不满足，就是假的，我们叫FALSE

就跟分类讨论一样，如果有一个条件，x>5，x 的值可能随着我们的数据一直变化，如果 x 是”6, 10, 20, 100” 这种数值, 它就是 真 ，如果 x 是 “-1, 0, 1, 5” 这种数值，那它的输出就是 假。

IF 函数

我先说这个玩意儿怎么理解，IF(条件, 值1, 值2)
根据我们一开始对于函数的理解，这个玩意儿也是传入一些东西，然后输出结果，
只是如果IF(条件, 值1, 值2)中的 条件 参数，是满足的，也就是 真 的时候，这个函数输出的结果就是 值1, 如果不满足就是 值2,

如有两个单元格 a 和 b, 假如 a 中是我们学校的分数, b 是一个函数：IF(x>90, '赢麻了', '输麻了') ,
在这个场景下，如果 a 中的值是比90大的，那么 y 中的值就是 '赢麻了', 当然如果 a 中的值小于等于90, 那么 b 中的值就会变为 '输麻了',

更贴近你们学习的内容的例子：IF(A1>3, B1-B2, B1+B2) 的意思就是，如果 “单元格 A1 中的值 比 3 大”，那么结果就是 “B1-B2”, 如果 “单元格 A1 中的值 比 3 小或相等”, 那么结果就是 “B1+B2”

也许你可以理解到，它是用于针对不同情况输出不同的值，尤其是”如果就，如果就”，”当……时，当……时”比如：
“如果是男生就输出boy，如果是女生就输出girl”

IF 函数嵌套

我们之前说，函数也可以作为参数，IF(条件, 值1, 值2)
假如有函数 a(参数) 和 b(参数), 我其实可以写成这样, IF(条件, 函数a, 函数b), 理解起来也很简单，就是条件==满足==(为真，也就是TRUE)时，单元格中的值就是 函数a 执行后的值，如果条件不满足(为假, 也就是FALSE)时，单元格中的值就是 函数b 执行后的值。

如: IF(A1>10, SUM(B1:B10), 0) 如果 单元格 A1 中的值比 10 大，那么结果就是 B1到B10 的和，如果 A1 的值不必10大，那么结果就是 0

还可以更进一步，IF(条件, 值1, 值2) 中的条件也可以存在函数

如: IF(COUNT(A1:A20)>10, SUM(A1:A20), 0) : 如果 A1到A20 中有值的单元格数量大于10，那么输出它们的和，如果不满足条件，就输出0

与或非

我看你们课本上只写了一个 AND ，我觉得一块说一下比较好理解？
不知道你们老师怎么说的这块地方，没有做我这种扩充的话，我多写的看看就行。

当你有多个条件需要判断时，如：
“18岁男生是成年男性”，就是 “年龄大于等于18 且 性别” ，这两个条件==都满足==才能是 “成年男性”, 有一个不满足都不行，这个就是 “与”，也就是 AND 函数
举例子:：A1是装有年龄的单元格，A2是装有性别的单元格，AND(A1>18, A2="男")，
A1的值为20(满足)，A2为”男”(满足)时，结果就是TRUE， A1的值为10(不满足)，A2为”男”(满足)时，结果就是FALSE

“某某某与狗不得入内”，就是 “某某某 或 狗” 都不得入内，这两个条件 ==只要满足一个== 就 “不得入内”, 有一个满足就可以，这个就是 “或”，也就是 OR 函数
举例子：A1是姓名，A2是是否为狗, 则可以写成 OR(A1="某某某", A2="狗")

“非” 就很简单了，NOT函数，NOT(TRUE) 就是 FALSE，NOT(FALSE) 就是 TRUE

当用 IF 时需要有多个条件进行判断时使用，如 “你是满22岁的男性我就跟你结婚，否则就分手！”
剖析一下，IF(条件, 值1, 值2), 条件为 “年龄>=22”且”性别为男”，如果都满足满足，就是 值1，这里则为 结婚函数，如果有一个不满足，就是值2，这里是 分手函数，因为两个条件都需要满足，我们需要使用AND作为 IF 中的 条件 根据分析我们可以写成：
A1是岁数，A2是性别，假如有结婚和分手的函数IF(AND(A1>=22, A2="男"), 结婚(), 分手())

之前没说的 SUMIF 和 COUNTIF

这俩哥们的作用跟 SUM, 和 COUNT 的作用一样，只不过就是 “如果满足条件才会参与到求和中” 和 “如果满足条件才会被计数”，

如(A1:A5)是一串数：-1, 0, 1, 2, 4

对于 =SUM(A1:A5) 结果就是 6 (-1+0+1+2+4)
对于 =SUMIF(A1:A5, ">0") 的结果就是 7 (1+2+4)

对于 =COUNT(A1:A5) 的结果就是 5
对于 =COUNTIF(A1:A5, ">0") 的结果就是 3
